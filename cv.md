# Code Example
 **1.Two Sum** <a href="https://leetcode.com/asldev/" target="_blank"><img align="center" src="https://raw.githubusercontent.com/rahuldkjain/github-profile-readme-generator/master/src/images/icons/Social/leet-code.svg" alt="https://leetcode.com/asldev/" height="30" width="40" /></a>
Given an array of integers `nums` and an integer `target`, return *indices of the two numbers such that they add up to* `target`.

You may assume that each input would have **exactly one solution**, and you may not use the same element twice.

You can return the answer in any order. 
```javascript
    var twoSum = function(nums, target) {
    let map = new Map();
    let diff;
    let cur;

    for(let i = 0; i < nums.length; i++){
        cur = nums[i];
        if(map.has(cur)){
            return [map.get(cur), i];
        }else{
            diff = target-nums[i];
            map.set(diff, i); 
        }

    }
};
```
